# Numi v1 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first shippable `numi` MVP from `docs/spec.md`: config discovery, `.xcassets` and `.strings` parsing, unified IR, deterministic rendering, built-in and custom templates, `generate`/`check`/`init`/`config`/`dump-context` commands, and actionable diagnostics.

**Architecture:** Start from a small Rust workspace so public boundaries are explicit from day one: `numi-cli` for CLI contract, `numi-config` for discovery/schema/validation, `numi-ir` for the typed graph and normalization, `numi-diagnostics` for stable error surfaces, and `numi-core` for parsing, context building, rendering, and output orchestration. Defer `.xcstrings`, partial templates, and aggressive caching until after the MVP works end-to-end with deterministic fixture coverage.

**Tech Stack:** Rust 2024, `clap`, `serde`, `toml`, `camino`, `miette`, `thiserror`, `ignore`, `xcassets`, `langcodec`, `minijinja`, `assert_cmd`, `insta`, `tempfile`, `fs_extra`, `criterion`.

---

## Scope Decisions Locked For This Plan

- `.xcstrings` is explicitly deferred to v1.5. v1 ships `.strings` support only, but keeps the input kind reserved so diagnostics can explain the deferral cleanly.
- Built-in templates generate their helper types in the same output file in v1. No external runtime dependency is required.
- Custom template partials/includes are out of scope for v1. A single template file is the only supported custom template shape.
- `swiftgen.toml` is the only discovered filename in v1.
- Shared parsed graphs are implemented as an internal optimization inside `numi-core`; there is no user-facing config knob for it in v1.
- Bundle handling is represented in the stable context model and consumed by built-ins; template helpers only cover string escaping and naming concerns.
- Swift formatting logic stays simple: structural layout belongs in templates, while escaping and naming belong in renderer filters.

pub use discovery::{discover_config, CONFIG_FILE_NAME};
pub use model::*;
pub use validate::validate;

use camino::{Utf8Path, Utf8PathBuf};

pub fn load_config(path: &Utf8Path) -> Result<Config, String> {
    let text = std::fs::read_to_string(path).map_err(|err| err.to_string())?;
    let config: Config = toml::from_str(&text).map_err(|err| err.to_string())?;
    validate(&config).map_err(|diagnostics| diagnostics.into_iter().map(|item| item.message).collect::<Vec<_>>().join("\n"))?;
    Ok(config)
}

pub fn resolve_jobs<'a>(config: &'a Config, selected_job: Option<&str>) -> Result<Vec<&'a JobConfig>, String> {
    match selected_job {
        Some(name) => config
            .jobs
            .iter()
            .find(|job| job.name == name)
            .map(|job| vec![job])
            .ok_or_else(|| format!("Unknown job `{name}`")),
        None => Ok(config.jobs.iter().collect()),
    }
}

pub fn resolve_config_path(start: &Utf8Path, explicit: Option<&Utf8Path>) -> Result<Utf8PathBuf, String> {
    discover_config(start, explicit).map_err(|diagnostic| diagnostic.message)
}

use camino::Utf8PathBuf;
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Config {
    pub version: u32,
    #[serde(default)]
    pub defaults: Defaults,
    pub jobs: Vec<JobConfig>,
}

#[derive(Debug, Clone, Default, Serialize, Deserialize)]
pub struct Defaults {
    #[serde(default = "default_access_level")]
    pub access_level: String,
    #[serde(default)]
    pub bundle: BundleConfig,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct JobConfig {
    pub name: String,
    pub output: Utf8PathBuf,
    pub inputs: Vec<InputConfig>,
    pub template: TemplateConfig,
    #[serde(default)]
    pub access_level: Option<String>,
    #[serde(default)]
    pub bundle: Option<BundleConfig>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct InputConfig {
    #[serde(rename = "type")]
    pub kind: InputKind,
    pub path: Utf8PathBuf,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum InputKind {
    Xcassets,
    Strings,
    Xcstrings,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TemplateConfig {
    pub builtin: Option<String>,
    pub path: Option<Utf8PathBuf>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct BundleConfig {
    #[serde(default = "default_bundle_mode")]
    pub mode: String,
    #[serde(default)]
    pub identifier: Option<String>,
}

fn default_access_level() -> String { "internal".into() }
fn default_bundle_mode() -> String { "module".into() }

impl JobConfig {
    pub fn access_level<'a>(&'a self, defaults: &'a Defaults) -> &'a str {
        self.access_level.as_deref().unwrap_or(defaults.access_level.as_str())
    }

    pub fn bundle_mode<'a>(&'a self, defaults: &'a Defaults) -> &'a str {
        self.bundle.as_ref().map(|bundle| bundle.mode.as_str()).unwrap_or(defaults.bundle.mode.as_str())
    }

    pub fn bundle_identifier<'a>(&'a self, defaults: &'a Defaults) -> Option<&'a str> {
        self.bundle
            .as_ref()
            .and_then(|bundle| bundle.identifier.as_deref())
            .or(defaults.bundle.identifier.as_deref())
    }
}

// crates/numi-config/src/discovery.rs
use camino::{Utf8Path, Utf8PathBuf};
use ignore::WalkBuilder;
use numi_diagnostics::Diagnostic;

pub const CONFIG_FILE_NAME: &str = "swiftgen.toml";

pub fn discover_config(start: &Utf8Path, explicit: Option<&Utf8Path>) -> Result<Utf8PathBuf, Diagnostic> {
    if let Some(explicit) = explicit {
        return Ok(explicit.to_owned());
    }

    for dir in start.ancestors() {
        let candidate = dir.join(CONFIG_FILE_NAME);
        if candidate.exists() {
            return Ok(candidate);
        }
    }

    let mut matches = Vec::new();
    let mut builder = WalkBuilder::new(start);
    builder.hidden(false);
    builder.filter_entry(|entry| {
        !matches!(
            entry.file_name().to_str(),
            Some(".git" | ".build" | "build" | "DerivedData")
        )
    });

    for result in builder.build() {
        let entry = result.map_err(|err| Diagnostic::error(err.to_string()))?;
        if entry.file_name() == CONFIG_FILE_NAME {
            matches.push(Utf8PathBuf::from_path_buf(entry.into_path()).unwrap());
        }
    }

    match matches.len() {
        0 => Err(Diagnostic::error(format!("No {} found", CONFIG_FILE_NAME))),
        1 => Ok(matches.remove(0)),
        _ => Err(Diagnostic::error(format!(
            "Multiple configuration files found under {}",
            start
        ))),
    }
}

// crates/numi-config/src/validate.rs
use crate::{Config, TemplateConfig};
use numi_diagnostics::Diagnostic;
use std::collections::BTreeSet;

pub fn validate(config: &Config) -> Result<(), Vec<Diagnostic>> {
    let mut diagnostics = Vec::new();

    if config.version != 1 {
        diagnostics.push(Diagnostic::error(format!("Unsupported config version {}", config.version)));
    }

    if config.jobs.is_empty() {
        diagnostics.push(Diagnostic::error("Config must declare at least one job"));
    }

    let mut names = BTreeSet::new();
    for job in &config.jobs {
        if !names.insert(job.name.clone()) {
            diagnostics.push(Diagnostic::error(format!("Duplicate job name `{}`", job.name)));
        }
        if job.inputs.is_empty() {
            diagnostics.push(Diagnostic::error(format!("Job `{}` must declare at least one input", job.name)));
        }
        match &job.template {
            TemplateConfig { builtin: Some(_), path: None } => {}
            TemplateConfig { builtin: None, path: Some(_) } => {}
            _ => diagnostics.push(Diagnostic::error(format!(
                "Job `{}` must set exactly one of template.builtin or template.path",
                job.name
            ))),
        }
    }

    if diagnostics.is_empty() { Ok(()) } else { Err(diagnostics) }
}

}

pub fn swift_identifier(input: &str) -> String {
    let mut result = input
        .split(|ch: char| !ch.is_ascii_alphanumeric())
        .filter(|segment| !segment.is_empty())
        .map(|segment| {
            let mut chars = segment.chars();
            match chars.next() {
                Some(first) => format!("{}{}", first.to_ascii_uppercase(), chars.as_str()),
                None => String::new(),
            }
        })
        .collect::<String>();

    if result.is_empty() {
        result.push('_');
    }

    if matches!(result.as_str(), "class" | "struct" | "enum" | "protocol") {
        return format!("`{result}`");
    }

    result
}

pub fn normalize_scope(job_name: &str, raw_entries: Vec<RawEntry>) -> Result<Vec<ResourceEntry>, Vec<Diagnostic>> {
    let mut diagnostics = Vec::new();
    let mut namespaces: BTreeMap<String, Vec<RawEntry>> = BTreeMap::new();

    for raw in raw_entries {
        let namespace = raw.path.rsplit_once('/').map(|(head, _)| head.to_string()).unwrap_or_default();
        namespaces.entry(namespace).or_default().push(raw);
    }

    let mut entries = Vec::new();
    for (namespace, group) in namespaces {
        let mut seen = BTreeSet::new();
        let mut children = Vec::new();

        for raw in group {
            let leaf_name = raw.path.rsplit('/').next().unwrap_or(&raw.path);
            let swift_identifier = swift_identifier(leaf_name);
            if !seen.insert(swift_identifier.clone()) {
                diagnostics.push(Diagnostic::error(format!(
                    "identifier collision in job `{job_name}` for `{}`",
                    raw.path
                )));
                continue;
            }

            children.push(ResourceEntry {
                id: raw.path.clone(),
                name: leaf_name.to_string(),
                source_path: raw.source_path,
                swift_identifier,
                kind: raw.kind,
                children: Vec::new(),
                properties: raw.properties,
                metadata: BTreeMap::new(),
            });
        }

        children.sort_by(|left, right| left.id.cmp(&right.id));

        if namespace.is_empty() {
            entries.extend(children);
        } else {
            let name = namespace.rsplit('/').next().unwrap_or(&namespace).to_string();
            entries.push(ResourceEntry {
                id: namespace.clone(),
                name: name.clone(),
                source_path: Utf8PathBuf::from("virtual"),
                swift_identifier: swift_identifier(&name),
                kind: EntryKind::Namespace,
                children,
                properties: BTreeMap::new(),
                metadata: BTreeMap::new(),
            });
        }
    }

    entries.sort_by(|left, right| left.id.cmp(&right.id));

    if diagnostics.is_empty() { Ok(entries) } else { Err(diagnostics) }
}

pub fn write_if_changed(path: &Utf8Path, contents: &str) -> Result<bool, String> {
    if let Ok(existing) = fs::read_to_string(path) {
        if existing == contents {
            return Ok(false);
        }
    }

    if let Some(parent) = path.parent() {
        fs::create_dir_all(parent).map_err(|err| err.to_string())?;
    }

    let parent = path.parent().unwrap();
    let mut temp = NamedTempFile::new_in(parent).map_err(|err| err.to_string())?;
    temp.write_all(contents.as_bytes()).map_err(|err| err.to_string())?;
    temp.persist(path).map_err(|err| err.to_string())?;
    Ok(true)
}

pub enum CheckOutcome {
    UpToDate,
    Stale(Vec<String>),
}

pub fn check(config_path: &Utf8Path, selected_job: Option<&str>) -> Result<CheckOutcome, String> {
    let rendered_jobs = render_jobs(config_path, selected_job)?;
    let mut stale = Vec::new();

    for rendered in rendered_jobs {
        match std::fs::read_to_string(&rendered.output_path) {
            Ok(existing) if existing == rendered.contents => {}
            _ => stale.push(rendered.output_path.to_string()),
        }
    }

    if stale.is_empty() {
        Ok(CheckOutcome::UpToDate)
    } else {

// helper used by crates/numi-cli/src/main.rs
fn resolve_config_path(args: &cli::JobSelection) -> miette::Result<camino::Utf8PathBuf> {
    let cwd = camino::Utf8PathBuf::from_path_buf(std::env::current_dir()?).unwrap();
    let explicit = args
        .config
        .as_ref()
        .map(|path| camino::Utf8PathBuf::from_path_buf(path.clone()))
        .transpose()
        .map_err(|_| miette::miette!("config path must be valid UTF-8"))?;
    numi_config::resolve_config_path(&cwd, explicit.as_deref()).map_err(|message| miette::miette!(message))
}
